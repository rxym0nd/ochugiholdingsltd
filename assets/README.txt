LOGO INSTRUCTIONS
-----------------
1. Drop your logo file into this folder
2. Name it exactly: logo.png
3. It will auto-appear in the sidebar across all pages
4. Recommended size: 36x36px to 72x72px (square works best)
5. PNG with transparent background looks best on dark background

If your logo is a different format (JPG, SVG, WebP):
- Rename the file to match
- Open js/nav.js
- Find: src="assets/logo.png"
- Change "logo.png" to your filename (e.g. "logo.svg")
